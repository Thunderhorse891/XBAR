# XBAR HORSE APP INSTALLATION

This is the full setup for running the X BAR Management app on Windows.

### Getting started
1. Clone repo: https://github.com/Thunderhorse891/xbar-horse-management-app
2. Run one of the scripts below:

- `cripts/download-and-install.ps1` - - Runs the latest installer from GitHub
- `cripts/local-build.ps1` - - Builds locally on your windows system

### Gained Access from GitHub Actions
-Runs whenever you push to main - creates a new auto-built.
-Releases an installer...
[Click to download latest build](https://github.com/Thunderhorse891/xbar-mg-app/releases)

### Windows Installation Guide
1. Run `download-and-install.ps1` script
2. Follow the installer prompts
3. App should appear in the System Tray

This is totally offline-first. No cloud.
